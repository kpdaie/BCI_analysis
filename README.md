# BCI_analysis

### Install
To use as a package: 
- create environment
- clone from github 
- enter to the root directory of this repo 
- and pip install the package:
```
conda create --name bci_analysis python=3.8
conda activate bci_analysis
git clone https://github.com/kpdaie/BCI_analysis.git
cd BCI_analysis
pip install -e .
```
